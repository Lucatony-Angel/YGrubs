# YGrubs App

This directory contains the React + TypeScript frontend for YGrubs.

For project overview, architecture, and internship-facing highlights, see the root README:

- `../README.md`

## Local Development

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start Vite dev server
- `npm run build` - Type-check and build production assets
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
